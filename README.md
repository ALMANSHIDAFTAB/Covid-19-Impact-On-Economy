# Covid-19-Impact-On-Economy
📊 COVID-19 Impact on Economy Analysis
📌 Project Overview

This project analyzes the impact of COVID-19 on the global economy using data analysis and visualization techniques. The pandemic caused a major economic slowdown worldwide, affecting GDP, employment, and multiple industries.

The goal of this project is to extract meaningful insights from real-world data and understand how different economic indicators were affected during the pandemic.

🎯 Objectives
Analyze the relationship between COVID-19 cases and economic indicators
Identify trends and patterns across countries
Understand the impact on GDP, deaths, and stringency index
Perform Exploratory Data Analysis (EDA)
Generate visual insights for better decision-making
🛠️ Tech Stack
Python
Pandas – Data manipulation
NumPy – Numerical operations
Matplotlib & Seaborn – Data visualization
Jupyter Notebook – Analysis environment
📂 Dataset

The dataset includes key variables such as:

Country and date
COVID-19 cases and deaths
GDP per capita
Population
Human Development Index
Stringency Index

👉 These datasets are commonly used to analyze pandemic impact and economic trends across countries.

🔍 Key Analysis Performed
Data Cleaning and Preprocessing
Exploratory Data Analysis (EDA)
Correlation analysis between COVID cases and economic indicators
Country-wise comparison
Trend analysis over time
📈 Key Insights
Countries with higher stringency measures showed controlled case growth
Significant impact observed on GDP across multiple countries
High population countries experienced varied economic effects
Pandemic impact varied significantly across regions
📊 Visualizations
Line plots for trend analysis
Bar charts for country comparison
Heatmaps for correlation analysis
